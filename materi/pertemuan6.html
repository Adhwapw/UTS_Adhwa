Materi 6: Perulangan

1. Contoh Program `while`

<?php
$i = 0;
while ($i < 10) {
    echo "Baris ke-" . $i . "<br>";
    $i++;
}
?>

2. Contoh Program `do-while`

<?php
$i = 0;
do {
    echo "Baris ke-" . $i . "<br>";
    $i++;
} while ($i < 10);
?>

3. Contoh Program `for`

<?php
for ($i = 0; $i < 10; $i++) {
    echo "Baris ke-" . $i . "<br>";
}
?>

4. Contoh Program `foreach`

<?php
$pegawai = array(
    'manajer' => 'Cecep',
    'dirut'   => 'Asep',
    'umum'    => 'Dudu',
    'teknisi' => 'Jaja'
);

foreach ($pegawai as $jabatan => $nama) {
    echo "Jabatan: " . $jabatan . ", Nama: " . $nama . "<br>";
}
?>

5. Contoh Program Menentukan Bilangan Genap dan Ganjil

<?php
echo "Bilangan Genap: <br>";
for ($i = 0; $i < 100; $i += 2) {
    echo $i . " ";
}

echo "<br>Bilangan Ganjil: <br>";
for ($i = 1; $i < 100; $i += 2) {
    echo $i . " ";
}
?>
