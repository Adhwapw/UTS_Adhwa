Materi 5: Mengontrol Alur Program

1. Contoh Program Statement `if`

<?php
$nilai = 75;
if ($nilai >= 70) {
    echo "Anda lulus!";
}
?>

2. Contoh Program `if` dengan Dua Kondisi

<?php
$umur = 20;
if ($umur >= 18) {
    echo "Anda sudah dewasa.";
} else {
    echo "Anda masih anak-anak.";
}
?>

3. Contoh Program `if-elseif-else`

<?php
$umur = 16;
if ($umur < 13) {
    echo "Anak-anak";
} elseif ($umur >= 13 && $umur <= 17) {
    echo "Remaja";
} else {
    echo "Dewasa";
}
?>

4. Contoh Program Statement `switch`

<?php
$hari = "Senin";
switch ($hari) {
    case "Senin":
        echo "Hari ini adalah Senin.";
        break;
    case "Selasa":
        echo "Hari ini adalah Selasa.";
        break;
    default:
        echo "Hari tidak dikenal.";
}
?>

5. Contoh Program Menentukan Bilangan Positif, Negatif, atau Nol 

<?php
$angka = -5;
if ($angka > 0) {
    echo "Bilangan positif";
} elseif ($angka < 0) {
    echo "Bilangan negatif";
} else {
    echo "Bilangan nol";
}
?>
