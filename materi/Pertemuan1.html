Materi 1: HTML Dasar

1. Heading dalam HTML

Berikut contoh penggunaan heading dalam HTML dari H1 sampai H6:

<h1>Ini adalah Heading H1</h1>
<h2>Ini adalah Heading H2</h2>
<h3>Ini adalah Heading H3</h3>
<h4>Ini adalah Heading H4</h4>
<h5>Ini adalah Heading H5</h5>
<h6>Ini adalah Heading H6</h6>

2. Paragraf dalam HTML

Untuk menambahkan paragraf dalam HTML, gunakan tag <p>:

<p>Ini adalah contoh paragraf dalam HTML.</p>

3. Text Alignment

kita dapat mengatur perataan teks dengan atribut text-align:

<p text-align="left">Teks ini rata kiri.</p>
<p text-align="center">Teks ini rata tengah.</p>
<p text-align="right">Teks ini rata kanan.</p>

4. Membuat Baris Baru

Gunakan tag <br> untuk membuat baris baru:

<p>Ini adalah baris pertama.<br>Ini adalah baris kedua.</p>

5. Format Superscript

Untuk membuat teks superscript, gunakan tag <sup>:

<p>Rumus matematika: E = mc<sup>2</sup></p>
